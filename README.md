# Claude Code // Vibe Coding Workshop

A beautifully designed, single-page workshop guide that teaches **anyone** — no coding experience needed — how to build apps using [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

https://hack.schwenk.tech/

> You talk. The computer builds.

## What Is This?

A ready-to-use interactive guide for running a **60-minute vibe coding workshop**. Hand it to participants, point them at a terminal, and let them build their first app by just *talking*.

The guide covers:

- **Setup** — three steps from zero to building
- **The Process** — a minute-by-minute timeline for a 60-minute session
- **Cheatsheet** — quick reference for common actions
- **Slash Commands** — power-user commands via the [Compound Engineering](https://github.com/nichochar/compound-engineering) plugin
- **Mindset Tips** — how to think like a director, not a programmer

## Quick Start

Just open `index.html` in a browser. No build step, no dependencies, no framework — it's a single self-contained HTML file.

```bash
# clone and open
git clone https://github.com/fsch/claude-code-quickstart.git
cd claude-code-quickstart
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

## Who Is This For?

- **Workshop organizers** running intro sessions on AI-assisted development
- **Curious non-coders** who want a structured first experience with Claude Code
- **Teams** introducing vibe coding to designers, PMs, or other non-engineering roles

## Design

Dark theme, monospace typography (JetBrains Mono + Syne), ambient glow effects, scroll-reveal animations — designed to feel like a dev tool, not a slide deck. Fully responsive for mobile.
